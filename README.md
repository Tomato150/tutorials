# pixi.js Tutorials

This is the source project, view the [HTML Site](http://peepsquest.github.io/tutorials).

## Building

    npm install projmate-cli@0.1.0-dev -g
    npm istall -d
    pm run all

## Adding another tutorial

*   Add markdown file  to `src`
*   Update `src/toc.md`
*   Rebuild

        pm run all
